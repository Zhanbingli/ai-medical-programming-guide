## 图像分割方法

### 1. 阈值分割

```python
import cv2

def threshold_segmentation(image, threshold=128):
    """阈值分割"""
    _, binary = cv2.threshold(image, threshold, 255, cv2.THRESH_BINARY)
    return binary

# Otsu自动阈值
_, otsu_binary = cv2.threshold(image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
```
