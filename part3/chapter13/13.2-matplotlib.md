# 13.2 Matplotlib基础绘图

## Matplotlib核心概念

```python
Figure(画布)
  └── Axes(坐标系)
       ├── X轴
       ├── Y轴
       ├── 标题
       ├── 图例
       └── 数据系列
```

## 基础绘图示例

### 1. 简单线图

```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.plot(x, y)
plt.xlabel('Time')
plt.ylabel('Signal')
plt.title('Simple Plot')
plt.grid(True)
plt.show()
```

### 2. 柱状图

```python
categories = ['A药', 'B药', 'C药']
values = [15.2, 10.8, 12.5]

plt.bar(categories, values)
plt.ylabel('疗效 (mmHg)')
plt.title('三种药物疗效对比')
plt.savefig('bar_chart.png', dpi=300, bbox_inches='tight')
```

### 3. 散点图

```python
x = np.random.randn(100)
y = 2*x + np.random.randn(100)*0.5

plt.scatter(x, y, alpha=0.5)
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Scatter Plot')
```
