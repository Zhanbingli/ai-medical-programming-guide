## 如何参与社区

### 1. 提问的艺术

```
❌ 糟糕的问题:
"为什么我的代码报错?help!"

✅ 好的问题:
标题: Pandas读取Excel时UnicodeDecodeError

描述:
环境: Python 3.9, pandas 1.5.0, Windows 10
问题: 读取Excel文件时报错
代码:
```python
df = pd.read_excel('data.xlsx')
```
错误信息:
```
UnicodeDecodeError: 'utf-8' codec can't decode...
```
已尝试: 改用read_excel(..., encoding='gb2312')仍报错
```

### 2. 回答问题

```
- 帮助新手
- 巩固知识
- 建立声誉
- 扩大人脉
```

### 3. 分享经验

```
- 写博客记录学习
- 分享踩坑经验
- 开源项目
- 组织线下活动
```

## 建立自己的社区

如果你所在医院/科室还没有:

```
1. 发起学习小组
   - 每周固定时间
   - 共同学习一个主题
   - 分享工作应用

2. 组织内部分享
   - 月度技术分享会
   - 展示实用工具
   - 解决共同问题

3. 建立资源库
   - 收集常用代码
   - 整理学习资料
   - 分享最佳实践
```
